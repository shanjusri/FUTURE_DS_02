# Customer Retention & Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn data to identify retention patterns, customer behavior, and the factors influencing customer churn using Power BI.

## Objectives
- Calculate overall churn rate
- Identify churn by contract type
- Analyze churn by gender
- Compare churn across payment methods
- Analyze churn by internet service
- Study customer tenure and monthly charges
- Provide actionable business insights

## Tools Used
- Power BI
- Microsoft Excel (CSV Dataset)

## Dashboard Highlights
- Total Customers
- Churned Customers
- Churn Rate
- Churn by Gender
- Churn by Contract
- Churn by Internet Service
- Churn by Payment Method
- Churn by Senior Citizen
- Average Monthly Charges by Churn
- Average Total Charges by Churn
- Customer Tenure Analysis

## Files Included
- Customer_Churn_Analysis.pbix
- WA_Fn-UseC_-Telco-Customer-Churn.csv

## Outcome
The dashboard helps identify customer segments with higher churn risk and provides insights that can support better customer retention strategies.
